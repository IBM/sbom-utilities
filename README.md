# SBOM Utilities

## Overview

This repository contains a collection of lightweight utilities and examples intended to support **Software Bill of Materials (SBOM) analysis**.

The tools in this repository are designed to help with common SBOM-related tasks such as:
- Inspecting and exploring SBOM content
- Performing basic validation and sanity checks
- Extracting or summarising information (e.g. components, licenses, dependencies)

Where possible, the utilities aim to work with widely adopted SBOM formats such as **CycloneDX** and **SPDX**.
